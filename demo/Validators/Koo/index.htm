使用说明

Koo.js标签验证插件基于Jquery开发，为KooTeam.com而生，取名Koo.js。使用十分简便，不需写任何代码即可实现各种的表单验证，同时支持select，checkbox，radio等默认值，一个标签全部搞定。
在线讨论：http://www.oschina.net/p/koojs
QQ交流群群：222516124

1.页面中引入jquery.js库和koo.js
2.加入如下代码
    <script type="text/javascript" />
        $(document).ready(function () { $('form').Koo(); });
    </script>

3.验证标签对象可以是id或class两个属性之一，验证标签以“-”符号分隔，第一位标识唯一性，无验证意义。例如：id="test-need-mail"：表示必填字段，且只能输入邮箱，id="test1-date-need"：表示输入框会自动加载日期控件，且输入值不能为空，也可以写成class="myClass test-need-mail"或class="myClass test1-date-need"

4.自定义提示，把需要提示的信息，覆盖到title属性上，在验证的时候便提示自定义的提示信息。例如：id="test-need" title="干嘛输入空值呢？"，在验证不通过时会提示：“干嘛输入空值呢？”，而不是默认的"不能为空"

5.修改提示样式，在你的css中给.kootip这个样式定义规则即可，默认样式 .kootip{color:red;line-height:16px;}

回调函数

callback：回调函数必须要有Boolean类型的返回值，否则回调会失败

        <script type="text/javascript">
        var callback = function () {
            alert('执行回调函数');
            return false;
        }
        $(document).ready(function () { $('form').Koo(callback); });
        </script>
